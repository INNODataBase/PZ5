# PZ5_10_236_Ковалев_Таланян
## Задачи:
1) Дописать интерфейс на PuQt для приложенного кода
2) Реализовать возможность поиска каллорийности продуктов по названию или коду
3) Доступ к бд через апи
За основу было взято руководство "PyQt6 для начинающих" из ДЗкЛК4-1710.
Создано desktop-приложение на PyQt6 с современным интерфейсом для поиска информации о продуктах таких как: название, бренд, каллории и т.п.
Программа поддерживает поиск по названию и поиск по штрихкоду товаров с фильтрацией по точному соответствию.
Приложение представлено в виде трёх панелей:
- Левая панель - выбор типа поиска и ввод запроса;
- Центральная панель - подробная информация о выбранном товаре с калорийностью;
- Правая панель - все найденные товары или похожие продукты в которых пользователь видит необходимую ему информацию по поиску товаров.
Есть две кнопки-переключателя (с названия на штрихкод) и кнопки для перехода к похожему товару.
